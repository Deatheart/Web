CAUT

代码单元自动测试工具

主页：http://lab205.org/caut/

CAUT (C Analysis and Unit Testing) 是一个以动态符号执行为基础的工具，在C程序单元测试级别自动生成测试数据。

安装

仅支持Linux

可以使用git下载源代码

$ git clone http://github.com/tingsu/caut-lib

或者下载源代码包：caut.tar.gz或caut.tar.bz2

1. 安装CAUT前，需要安装Ocaml

在Ubuntu/Debian系发行版下：

$ sudo apt-get install ocaml

2. 编译源码

1) 编译c-ocaml-sqlite-lib

第一步：

$ cd ./tools/c-ocaml-sqlite-lib

第二步：

 $ make clean 
 $ make 
 $ sudo make install

库文件默认安装路径是 /usr/lib/ocaml

2) CAUT需要CIL。CIL的源码可以在http://kerneis.github.com/cil/doc/html/cil/获得。

3) 把lpsolve库所在目录加入PATH变量中。在 $(HOME)/.bashrc 里面添加如下内容

export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:/work/caut-lib/lpsolve
export C_INCLUDE_PATH=$C_INCLUDE_PATH:/work/caut-lib
