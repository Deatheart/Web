Rodin Event B

实时建模工具

主页：http://www.event-b.org/

Rodin工具支持对Event-B形式化方法的应用，它对Event-B模型语法分析与基于证明的验证提供核心的功能。B方法是由Jean-Raymond Abrial 在20世纪90年代初开发的，主要是支持软件的形式化开发，Event-B不仅仅只对软件组件进行建模，而且也对系统进行建模。Event-B主要是对系统进行建模和论证的，这些系统包含了许多物理组件，电子器件和软件。

Event B 在系统层面建模，并应用形式化方法进行分析。其主要的功能是，利用集合论作为建模符号，精细的区分系统不同层面的抽象，并运用数学的方法来证明系统不同层面细化的一致性。

安装

需要Java 1.6。为Windows，提供编译好的二进制程序下载。

Windows

下载Core Rodin Platform rodin-3.0-win.x86.zip 或 rodin-3.0-win.x86_64.zip

解压后运行rodin文件夹下的rodin.exe程序
